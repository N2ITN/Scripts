List of useful resources for getting started with Python

When in doubt, Google and stackoverflow are your friends.

Download open-source software
•	Download Python - Prefer 2.7 as it has the most compatiblity with third party modules.
	o	https://www.python.org/downloads/
	
•	Sublime Text 3 - My go-to IDE. It is quite simple, which can be either refreshing and zenlike or feel unfurnished, depending on the scope of your project. Useful for smaller scripts.
	o	http://www.sublimetext.com/3
	
•	Spyder - Great for scientific computing, and larger projects. Advanced features like automatic code completion, variable explorer, etc. Loaded with numerical and scientific modules. Can weigh down system performance.
	o	https://pythonhosted.org/spyder/


Learn to code in Python
•	Great interactive primer on Python data structures, operations, functions, etc
	o	http://www.codecademy.com/en/tracks/python

•	Online book for useful automation programming 
	o	http://automatetheboringstuff.com
	
•	Tutorials point - There are a plethora of useful resources online for python syntax, but this site in particular has clear, concise explanations of common syntax, with easy to understand code examples
	o http://www.tutorialspoint.com/python/index.htm

•	Reddit subs - /r/
	o	learnprogramming, python

Useful Modules
•	Download python libraries: 
	o	https://pip.pypa.io/en/latest/installing.html
			Once PIP is set up, open windows command processor enter: cd C:/python27/scripts then enter :  
			pip install module_name
			It will find the package & install it and it will be ready to use instantly

•	Read, write Excel docs
	o	http://www.python-excel.org/

•	Make HTTP requests online
	o	http://docs.python-requests.org/en/latest/

•	Data visualization (requires NumPy module) A mainstay of scientific computing, and part of the SciPy package.
	o	http://matplotlib.org/

•	Print pretty tables in your IDE
	o	https://code.google.com/p/prettytable/wiki/Tutorial
